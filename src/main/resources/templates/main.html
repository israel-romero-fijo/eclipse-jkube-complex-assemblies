Eclipse JKube, Quarkus and Qute

If you deployed this in a Kubernetes compatible cluster using Eclipse JKube (mvn package k8s:build k8s:resource k8s:apply) you'll be able to see all of the JKube processed fragments.

If you are running this application in your local environment (mvn quarkus:dev) you'll only be able to see the first fragment and a few errors indicating that some files are not available.
